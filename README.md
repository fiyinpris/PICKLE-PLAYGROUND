# PICKLE-PLAYGROUND
Pickle Playground is an innovative web and mobile platform designed to make learning math and computer programming an enjoyable experience for kids. Through interactive games and puzzles, Pickle Playground aims to cultivate computational thinking and problem-solving skills in a fun and engaging way.

Modes of Learning
Play Mode
Build-Code Game
In this mode, beginners can construct virtual structures using a building-block approach similar to Lego. The magic happens as the code is generated in real-time, allowing users to visually see the correlation between their creations and the underlying code.

Code-Build Game
For more advanced users, the Code-Build game provides a hands-on experience where users write code to play games or solve puzzles. This mode encourages a deeper understanding of programming concepts by blending coding challenges with interactive gameplay.

Create Mode
Unlocking creativity, Create Mode empowers users to become game designers. They can:

Design Games: Create custom games that align with their interests and learning objectives.
Craft Puzzles: Develop challenging puzzles to stimulate critical thinking.
Simulate Scenarios: Build interactive simulations for a more immersive learning experience.
